Help Desk Aprendizagem de desenvolvimento

Olá! Seja bem vindo ao sistema Help Desk desenvolvido por Francisco José feito através de estudos e cursos online.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular 12 e Spring Boot 2.
O sistema está rodando na nuvem sendo o front na Vercel e o back na Heroku.
O projeto tem como objetivo principal aprendizagem dos conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
